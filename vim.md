# VIM


#### 跳转
| 命令       | 解释                                               |
| ---------- | -------------------------------------------------- |
| `10G`      | 可以跳转到第10行 (`G`跳转至最后、`gg`跳转至最开头) |
| `Ctrl-w-w` | 在多个内部窗口之间跳转                             |
|            |                                                    |
|            |                                                    |
|            |                                                    |
